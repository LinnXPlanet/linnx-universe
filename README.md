## 🧩 Architecture

This site is AI-native in three layers:

1. **Visual layer** — Human-facing: deep space navy + Côte d'Azur azure + ultra-thin typography (SpaceX × Apple minimalism)
2. **Protocol layer** — Agent-facing: JSON-LD + llms.txt + callable skills as API-like endpoints
3. **Self-iteration layer** — Data-driven: update `llms.txt` and skills metadata, site content follows

## 🎨 Design Philosophy
